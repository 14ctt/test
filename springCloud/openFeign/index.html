# 自定义URL

原文 (opens new window)
ToB 场景，需要对每个商户自定义的 URL 进行调用。
只要接口中声明了 URI 这个参数就能自定义。

 @FeignClient(name = "dynamic")
 interface DynamicClient {
  @GetMapping("/")
  String get(URI uri);
 }
 
 dynamicClient.get(URI.create("https://github.com"));

# 异常

执行后会抛出负载均衡的异常：

java.lang.RuntimeException: com.netflix.client.ClientException:
Load balancer does not have available server for client: github.com

# 代码修改

@FeignClient 新增一个url参数即可，url任意值即可

 @FeignClient(name = "dynamic", url="http://www.baidu.com")
 interface DynamicClient {
  @GetMapping("/")
  String get(URI uri);
 }
 
 dynamicClient.get(URI.create("https://github.com"));
